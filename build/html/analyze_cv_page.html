analyze_cv_page module¶

analyze_cv_page.app(cv_text, requirements_text)¶

Analyze a CV and job requirements using a multi-step graph-based workflow.

Parameters:

  • cv_text (str) – Text content of the uploaded CV.

  • requirements_text (str) – Text content of the uploaded job requirements.

  • label (str) – Label for the Streamlit file uploader.

  • key (str) – Unique key for the uploader widget.

Returns:

Extracted text content from the uploaded .docx file.

str

analyze_cv_page.show()¶

Display the Streamlit UI to upload a CV and job requirements and perform their analysis.

This function creates a two-column layout where the user can upload the CV and the job requirements. When the “Analyze” button is pressed, the uploaded files are processed.
